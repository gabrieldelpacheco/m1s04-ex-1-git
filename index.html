Comando basicos Github

Comandos basícos para um repositório novo:

  1. git init (Iniciando um novo repositório)
  2. git add . (esta adicionando todos os arquivos)
  3. git commit -m "primeiro comentario" (Onde faz o comentário antes de enviar os arquivos.)
  4. git branch -M main (Renomeando a branch atual para main)
  5. git remote add origin https://github.com/gabrieldelpacheco/"nome_da_branch" (Adicionar o repositório remoto ao repositório do Git local. )
  6. git push -u origin main (Usado para enviar as alterações do ramo local 'main'.)

Comando para mudar de uma branch para a outra:

  • git checkout nome_da_branch_a_mudar
